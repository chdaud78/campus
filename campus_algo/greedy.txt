# 탐욕 알고리즘
- 현재 상황에서 당장 가장 좋아 보이는 상황만을 선택
- 그리디 알고리즘, 탐욕법이라 함
- 최적의 해를 구하기 위한 근사적인 방법으로 사용될 때가 많다.
- 현실 세계에서는 단순 탐욕 알고리즘으로는 최적의 해를 놓칠 수 있다.
- 최적의 해에 가까운 답을 뱉는 것만 고려하면 현실에서의 다양한 프로그램에서 "근사의 해"를 구하는 목적으로 사용되곤 한다.

# 코테에서 탐욕 알고리즘
- 채점 시스템은 시험 참가자의 코드 결과가 정해진결과와 같은지 확인
- 탐욕 알고리즘 유형에서는 탐욕 방법으로 최적의 해가 보장되는 문제가 출제됨.

# 접근 방법
1. 방법 고안 : 현재 상황에서 어떤 것을 선택할지 알고리즘을 고안
2. 정당성 확인 : 자신이 고안한 알고리즘이 항상 최적의 해를 보장하는지 확인
ex) 거스름 돈 문제 : 동전 갯수 최소값 -> 화폐 단위가 큰 거 부터 거슬러 주기
   ㄴ 화폐 단위가 "배수 관계"에 속하기 때문에 해결 가능
